# Stripe Payment Gateway Setup for VERA

1. Create Stripe account (stripe.com)
2. Get test/live keys from dashboard
3. Set STRIPE_SECRET_KEY and STRIPE_PUBLISHABLE_KEY in .env (never commit)
4. pip install stripe flask
5. Run `python payment/checkout.py` or integrate into main app

Honest pricing enforced server-side. License keys delivered post-payment.
